# PokéJava ![](https://img.shields.io/badge/version-To_be_released-blue?style=flat-square)
A Java 16+ wrapper for the PokéAPI.  

### Release
Version 0.1.0 will be released soon, and will support the following:
- Pokémons
- Pokémon species
- Formes
- Sprites
- Habitats
- Evolutions
- Stats
- Abilities
- Types
- Moves
- Genders
- Natures
- Growth rates

Everything else will follow in future updates.

## Built with
- [Jackson](https://github.com/FasterXML/jackson) - JSON library
- [Retrofit](https://github.com/square/retrofit) - type-safe REST/HTTP client
- [OkHttp](https://github.com/square/okhttp) - HTTP client
